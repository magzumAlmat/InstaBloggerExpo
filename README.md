## ✨ Features

- ✅ User authentication & profile management
- ✅ Category-based blogger/brand discovery
- ✅ Real-time feed with posts and stories
- ✅ Push notifications
- ✅ Cross-platform (iOS & Android)

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ (or Python 3.8+ for Python projects)
- npm or yarn



### Installation

```bash
# Clone the repository
git clone https://github.com/magzumAlmat/InstaBloggerExpo.git

# Navigate to the project
cd InstaBloggerExpo

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Run the application
npm start
```
